# 终端（Terminal）

思源插件，在侧边栏（Dock）中嵌入终端界面，基于 [xterm.js](https://github.com/xtermjs/xterm.js) 与 [node-pty](https://github.com/microsoft/node-pty)。

> 说明：当前版本只是 Vibe Coding 的 demo，用来验证想法，并非成熟产品。后续也许会增加更多功能，也可能调整方向；使用时请抱持这一预期。

## 环境要求

- 需在 Windows、Linux 或 macOS 上使用思源客户端；作者仅在 Mac 上开发与使用，尚未在其他系统实测。  
- 客户端版本不低于集市或插件详情页所标注的最低兼容要求。
